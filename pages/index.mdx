# Documentation

Supaboard is an open-source dashboard tool for Supabase and Postgres that allows you to easily create dashboards and charts from your data.
Our mission is to make it easy for anyone to create dashboards and admin panels with minimal effort.

Learn how to get started, explore our guides and check out how to make the most of your Supaboard dashboards.
